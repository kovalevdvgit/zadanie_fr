
Для запуска сервера нужно перейти  в виртульную среду прописав -> venv\Scripts\activate и запустить файл start из командной строки -> python start.py
в файли start.py можно запустить как отладочную версию -> 1, так и рабочую на uvicorn -> 2 (раскомментируйте нужный вариант)

API доступно по адресу -> http://localhost:8000/api/...
